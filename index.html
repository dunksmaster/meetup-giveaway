📋 Konfiguro Google Sheet

Bëhet një herë — zgjat 3 minuta. Kjo lejon regjistrimet nga telefonat të shfaqen live tek ti.

1

Krijo Google Sheet të ri

Shko te sheets.new dhe krijo spreadsheet të ri. Emërojë Giveaway Registrations.

2

Hap Apps Script

Kliko Extensions → Apps Script. Fshi kodin ekzistues dhe paste kodin e mëposhtëm:

function doPost(e){
  var s=SpreadsheetApp.getActiveSheet();
  var d=JSON.parse(e.postData.contents);
  s.appendRow([new Date(),d.name,d.email]);
  return ContentService.createTextOutput(JSON.stringify({ok:true})).setMimeType(ContentService.MimeType.JSON);
}
function doGet(e){
  var s=SpreadsheetApp.getActiveSheet();
  var rows=s.getDataRange().getValues();
  var data=rows.slice(1).map(function(r){return{time:r[0],name:r[1],email:r[2]};});
  return ContentService.createTextOutput(JSON.stringify({entries:data})).setMimeType(ContentService.MimeType.JSON);
}
3

Deploy si Web App

Kliko Deploy → New Deployment → Web App. Te Execute as: Me. Te Who has access: Anyone. Kliko Deploy dhe autentifiko.

4

Kopjo URL-në dhe paste këtu

Pas deploy, kopjo URL-në që fillon me https://script.google.com/macros/s/... dhe paste te hapi i mëparshëm.
